Unix and zshell notes

This page contains notes about ZShell

Last command with !!

$ ls /tmp/
$ !!file # Becomes ls /tmp/file

$! is the last argument from the previous command

$ ls /tmp/
$ cat $!/file # appends /file to last argument

Remeber that > overwrites and >> appends

ZShell for loop 
 $ for file in *
 for> do
 for>   echo $file
 for> end
or 
 $ for file in *
 for> echo $file
and output can be redirected: 
$ for file in *
for> do
for>   echo $file
for> done | sort > /tmp/hi

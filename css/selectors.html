Selector Cheatsheet

Element selector

Use the name of the element 
ul { ... }
Matches all ul's in the document, like: 
<ul>
  <li>Bod Dylan</li>
  <li>Mark Knopfler</li>
</ul>

Class selector

Declared with a dot followed by the id name 
.notes { ... }

<div class="notes">
</div>

Id selector

Declared with a hash followed by the id name 
#song { ... }

<div id="song">
</div>
Note: To be used sparingly. Very high specifity.

Descendant Combinator selector

Allows you to select descendants of a particular element. First specify the parent, followed by a space and the child elements to be selected. They do not need to be direct descendants 
#song ul { ... }

<div id="song">
    <div>
      <ul>...</ul>
    </div>
</div>
Note: To be used sparingly. Makes styles difficult to reuse

Child Combinator selector

Allows you to select direct descendants of a particular element. First specify the parent, followed by a greather-than and the child elements to be selected. They do not need to be direct descendants 
#song > ul { ... }

<div id="song">
  <ul>...</ul>
</div>

General Sibling Combinator selector

Select siblings (at the same level) of an specific element. Use tilde to separate the element whose siblings we want to select, and the siblings. 
h1 ~ p { ... }

<h1>Title</h1>
<p>Paragraph</p>

Adjacent Sibling Combinator selector

Select adjacent siblings of an specific element. Use plus to separate the element whose siblings we want to select, and the siblings. 
h1 + p { ... }

<h1>Title</h1>
<p>Paragraph</p>

Attribute selector

Select elements with a given attribute 
input[type='submit'] { ... }

<input type="submit">

Pseudo class selector

a:hover { ... }

Pseudo elements selector

h2:after

Selector chain

h1.banner

Multiple selectors in a single declaration

h1, h2, h3 { ... }
